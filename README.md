#  Cozy Bean Cafe

Welcome to **Cozy Bean Cafe** — a beautifully designed single-page café homepage built entirely using **HTML and CSS**.  
This project was created for the **"Web Design Project — No JavaScript" Hackathon**, focusing on creativity, design aesthetics, and clean code structure.

---
##  Live Demo  
View Project:   http://cozy-bean-cafe.netlify.app/ 
-----

##  About the Project

Cozy Bean Cafe captures the warmth and charm of a modern coffee shop.  
The design blends smooth color tones, elegant typography, and engaging hover animations to deliver a visually relaxing browsing experience — all without using JavaScript.

This project demonstrates how **CSS alone** can be used to create beautiful, interactive, and responsive designs.

---

## Features

- Elegant single-page cafe layout .
- Fully responsive design (works on mobile, tablet, and desktop) . 
-  Smooth hover transitions and animations using pure CSS . 
-  Sections: Home, Menu, About Us,Gellary and Contact . 
- Clean, semantic HTML5 structure and organized CSS.

---

## Tech Stack

- **HTML5** — for structure.  
- **CSS3** — for styling and animations . 
- **Google Fonts** — for modern typography  .
- **Unsplash images** — for beautiful visuals.
